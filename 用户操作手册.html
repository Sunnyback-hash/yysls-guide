一、软件概述

《叠叠消消乐》（英文名：TapCrush）是一款面向移动端的休闲消除类小游戏，适用于抖音小程序平台。游戏借鉴了经典堆叠三消玩法并加以创新，玩家通过点击选取叠放在不同层中的图标砖块，集齐三个相同图标即可消除得分。

本软件的主要特色包括：

  • 环形叠层布局：图标砖块以环形放射状排列，不同层使用不同的渐变色区分，视觉清晰通透。
  • 能量爆发系统：每完成一次消除积攒能量，能量满时自动清除一个被压住的砖块，给予玩家策略性回报。
  • 30关渐进难度：从3层30块到5层60块，难度平滑递增。
  • 激励广告系统：道具用尽或游戏失败时，可通过观看广告获取额外机会。
  • 音效反馈：使用 Web Audio API 实现即时音效，无需外部音频文件。

三、安装与启动

3.1 抖音小程序版

  1. 打开抖音 App，确保已更新至最新版本。
  2. 在搜索栏输入"叠叠消消乐"。
  3. 点击搜索结果中的小程序卡片进入游戏。
  4. 首次加载后即可开始游戏，无需注册或登录。

3.2 Web 版本

  1. 在移动设备或桌面浏览器的地址栏中输入游戏 URL。
  2. 页面加载完成后自动进入开始画面。
  3. 建议使用竖屏模式以获得最佳体验。

注意：本软件为单页应用（SPA），所有资源均内置于一个 HTML 文件中，无需安装任何额外插件或依赖。

四、主界面说明

4.1 开始画面

启动游戏后进入开始画面，包含以下元素：

  • 游戏图标：中央显示游戏 LOGO。
  • 游戏标题：显示"叠叠消消乐"及英文副标题 "TapCrush"。
  • 开始按钮：点击后进入游戏画面，从第1关开始。
  • 统计信息：显示最高分、总局数、已通关数。
  • 辅助按钮：包含"玩法说明"和"音效开关"按钮。
  • 版本号：页面底部显示当前软件版本。

4.2 游戏画面

游戏画面从上到下依次为：

  • 顶部栏：左侧显示游戏名称，右侧显示当前关卡(Lv.X)、得分、暂停按钮。
  • 能量条：显示当前能量积攒进度（0/3），满格触发能量爆发。
  • 游戏盘面：中央区域，显示环形叠层排列的图标砖块。未被遮盖的砖块可点击，被遮盖的砖块呈灰色不可操作。
  • 底部槽位：最多容纳7个图标，集齐3个相同图标自动消除。
  • 道具栏：包含撤回、洗牌、移出三个道具按钮。

4.3 暂停画面

点击顶部暂停按钮（⏸）后弹出暂停菜单，包含：

  • 继续游戏：返回游戏画面。
  • 返回主菜单：退出当前对局，回到开始画面。

五、游戏操作说明

5.1 基本玩法

  1. 观察盘面中叠放的图标砖块，上层砖块会遮盖下层砖块。
  2. 点击未被遮盖的砖块（显示为彩色且无灰色遮罩），该砖块会移动到底部槽位中。
  3. 当槽位中集齐 3个相同图标 时，自动消除并加100分。
  4. 消除时产生粒子飞溅特效和得分飘字。
  5. 消除所有盘面砖块即为过关。
  6. 槽位满7个且未触发消除时，游戏失败。

5.2 能量系统

能量系统是本游戏的核心创新机制：

  • 每成功消除一组（3个相同图标），能量槽增加1点。
  • 能量槽最多积累3点。
  • 能量满3点时自动触发"能量爆发"：随机清除一个被压住的砖块，并额外增加50分。
  • 能量爆发时有特殊视觉和音效反馈。

5.3 游戏结果

  • 过关：弹出胜利弹窗，显示本关得分。可选择"下一关"继续挑战。
  • 失败：弹出失败弹窗，显示本关得分。可选择"看广告复活"或"重新开始"。
  • 全部通关：完成第30关后弹出通关祝贺，可选择重新从第1关开始。

六、道具系统

游戏提供三种辅助道具，每种道具每局可免费使用一次，用尽后可通过观看广告再次获取。

道具图标功能说明
撤回 ↩ 撤销上一次移动操作，将最近移入槽位的砖块放回盘面原位置。适用于误操作挽回。
洗牌 🔀 随机重新排列盘面上所有剩余砖块的图标类型，有可能改变局势。适用于卡关时尝试新的排列组合。
移出 ✨ 从盘面上移除3个砖块（优先移除被压住的砖块）。适用于槽位紧张时需要快速减少盘面砖块数量。

6.1 免费使用

每局游戏中，每个道具首次点击可直接免费使用，使用后按钮变为半透明状态并显示📺标记。

6.2 广告续用

免费次数用尽后，点击已使用的道具按钮，弹出确认对话框，选择"看广告获取"即可通过观看激励视频广告获得一次额外使用机会。广告播放完成后道具效果立即生效。

七、广告系统

7.1 激励广告场景

本软件在以下场景提供激励视频广告：

场景奖励内容是否可选
道具用尽后点击道具按钮获得该道具的一次额外使用是（用户点击确认后播放）
游戏失败时复活继续游戏（移除槽位中3个砖块）是（用户点击"看广告复活"后播放）

7.2 广告播放流程

  1. 用户点击触发广告的按钮（道具按钮或复活按钮）。
  2. 弹出确认对话框，告知用户即将观看广告及奖励内容。
  3. 用户确认后，展示广告播放画面（含进度条和倒计时）。
  4. 广告播放完毕，自动关闭广告画面，奖励立即生效。
  5. 用户也可在确认阶段选择"取消"，放弃本次广告观看。

技术说明：当前版本使用模拟广告流程（2秒倒计时），接入抖音小程序时需替换为真实的激励视频广告 SDK 调用。

八、关卡系统

8.1 关卡结构

游戏共设 30关，难度逐关递增。递增维度如下：

关卡区间叠层数砖块总数图标种类
Lv.1 - Lv.53层30 - 33块10 - 11种
Lv.6 - Lv.154层36 - 45块12 - 15种
Lv.16 - Lv.305层45 - 60块15 - 20种

8.2 关卡生成算法

每关的盘面布局由程序自动生成，保证每次进入都有不同的排列。生成算法考虑以下因素：

  • 总砖块数随关卡递增（Lv.1约30块，Lv.30约60块）。
  • 叠层数在Lv.6增至4层，Lv.16增至5层。
  • 每层砖块呈环形排列，外圈大、内圈小，不同层使用不同颜色渐变区分。
  • 每层内部的图标类型随机排列，每种图标恰好出现3次。

8.3 关卡进度

通过当前关卡后自动进入下一关。完成第30关即全部通关，显示通关祝贺并可选择重新开始。
